# NovaBridgeDemo

Minimal sample project scaffold for NovaBridge demos and distribution bundles.

## Launch

```bash
UnrealEditor /path/to/NovaBridgeDemo/NovaBridgeDemo.uproject -RenderOffScreen -nosplash -nosound -unattended -nopause -NovaBridgePort=30010
```

Use this project when running headless services so UE does not open the project browser.
