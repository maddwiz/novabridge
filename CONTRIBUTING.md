# Contributing

NovaBridge is distributed as proprietary software.

## External Contributions

- Public pull requests are currently **not accepted** for direct merge.
- Bug reports and feature requests are welcome via GitHub Issues.
- Security reports should follow the contact path in `SUPPORT.md`.

## Internal Development

- All code changes should include validation evidence (tests, smoke runs, or build output).
- Keep module boundaries intact (`NovaBridgeCore`, `NovaBridge`, `NovaBridgeRuntime`).
- Update `CHANGELOG.md` and relevant docs when behavior or release workflows change.
